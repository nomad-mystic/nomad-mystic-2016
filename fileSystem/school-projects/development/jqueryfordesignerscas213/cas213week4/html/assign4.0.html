What if the user doesn't have JavaScript enabled?

You have two options for users with JavaScript disabled - which you might choose depend on the content of you page.

You might just leave the page as it is - and make sure the <dt> tags are styled in a way that makes them stand out and easy to pick up when you're scanning down through the page. This would be great solution if you anwsers are relatively short.

If your FQA page has long answers, it might be helpful to put a table of contents list to individual questions at the top of the page so users can click to jump directly to the question and answer they're interested in. This is similar to what we did in the tabbed example, but in this case, we would use jQuery to hide the table of contents when the page loaded since users with JavaScript wouldn't need to see table of content.
